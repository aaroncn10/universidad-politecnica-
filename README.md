# El Pueblo Tikuna — Landing page

Página web de la monografía **"El pueblo Tikuna: identidad transfronteriza, bilingüismo y la fiesta de la Pelazón como ritual de paso"**.
Curso de Comunicación · Facultad de Ingeniería · Universidad Politécnica · Lima, 2026.

**Integrantes:** Aaron Collado · Milagros Vasquez · Fray Flores · Alan Espinoza · Jaime Callañaupa

## Cómo publicarla con GitHub Pages

1. Sube todos los archivos de esta carpeta a la raíz del repositorio `aaroncn10/universidad-politecnica` (arrastra los archivos en github.com → *Add file → Upload files*).
2. En el repositorio: **Settings → Pages → Source: Deploy from a branch → Branch: main / (root) → Save**.
3. En 1–2 minutos la web quedará en línea en:
   `https://aaroncn10.github.io/universidad-politecnica/`

## Archivos

- `index.html` — toda la página (una sola página, secciones ancladas)
- `styles.css` — estilos (paleta achiote/huito, tipografía Fraunces + Inter)
- `main.js` — animaciones (JS vanilla, sin dependencias)
- `lib/manifest.js` — datos del equipo
- `.htaccess` — cabeceras de caché (solo para hosting Apache tipo Hostinger; GitHub Pages lo ignora)

Sin frameworks, sin build: también funciona abriendo `index.html` directamente en el navegador.
